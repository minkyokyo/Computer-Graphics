# 이동  
마우스 좌표를 따라 cube에 transform 적용 
마우스 좌표를 따라 카메라에 transform 적용 

# 회전  
처음 마우스 클릭좌표 m1  
drag 후 클릭 좌표 m2
특정위치->m1, 특정위치 -> m2로 가는 벡터를 구함.  
두 벡터의 회전각을 구하고 외적을 이용하여 회전축을 구함
